# xiakankan
项目简介

龙虾数据看板是一个高颜值、玻璃态风格的 QQ 社交聚合工具前端原型。它解决了 QQ 原生功能中动态、群聊、文件、点赞、提醒等模块各自独立、频繁切换的痛点，将分散的社交数据重组为一个以用户为中心的驾驶舱。
核心设计理念：社交聚合 —— 文件标注来源、点赞溯源关系链、群消息直达回复。
视觉风格：石墨黑背景 + 霓虹青绿/蓝紫渐变 + 玻璃态毛玻璃卡片。

核心功能模块

 模块 -功能亮点:
| 首页顶部看板 | 5 个核心数据卡片（动态好友、提及、文件、点赞、@我）。点击卡片可平滑跳转至对应区域，实现“总览即导航”。 |
| 空间动态分组| 按 亲友/同学/好友/群友 四大关系链聚合动态。展开预览前 3 条，支持直接点赞，无需跳转 QQ 空间。 |
| 文件智能管理 | 6 大分类 + 文件来源标注（“来自XX群/好友”）。支持按类型/来源搜索、一键接收全部待收文件、自定义下载路径。 |
| 点赞互动中心 | 累计/今日点赞数 + 来源占比。一键全部回赞。点赞记录带头像，点击头像可查看该好友今日动态（社交溯源）。 |
| 会员管理| 等级展示、有效期倒计时（到期前 7 天橙色高亮预警）、一键续费模拟、核心权益罗列。 |
| 群消息聚合 | 群公告未读角标、@我总量、提及消息预览。预览区支持模拟快捷回复，聚合处理群聊互动。 |
| 数据刷新规则 | 顶栏实时倒计时，每日 00:00 自动清零今日数据（模拟）。 |
| 社交快捷入口 | 右下角悬浮按钮：快速发起群聊、快速回复好友、查看今日社交汇总。 |
